# Sajid LMS

A Learning Management System (LMS) project for educational content management and course delivery.

## 📋 Prerequisites

Before running this project, ensure you have the following installed:
- **Git** - [Download](https://git-scm.com/)
- **Node.js** (v14 or higher) - [Download](https://nodejs.org/)
- **npm** or **yarn** - Comes with Node.js

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/mdsajid-ui/Sajid-lms.git
cd Sajid-lms
```

### 2. Extract the Project Files

The project is packaged in `LMS-main.zip`. Extract it:

```bash
unzip LMS-main.zip
cd LMS-main
```

Or use your file explorer to extract the zip file.

### 3. Install Dependencies

```bash
npm install
```

Or if you prefer yarn:

```bash
yarn install
```

### 4. Start the Development Server

```bash
npm start
```

Or with yarn:

```bash
yarn start
```

The application should now be running. Open your browser and navigate to:
- **Local**: `http://localhost:3000` (or the port specified in your console)
